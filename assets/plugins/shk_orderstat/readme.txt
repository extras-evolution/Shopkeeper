
SHKorderStat 1.0

Плагин для MODx 1.x + Shopkeeper 1.x

Статистика заказов

--------------------------------------------------

http://modx-shopkeeper.ru/  

--------------------------------------------------

Установка:

1. Поместите папку shk_orderstat в папку с плагинами assets/plugins/.

2. В системе управления откройте "Элементы" -> "Управление элементами" -> "Плагины" -> "Создать плагин".

3. Название плагина:
   SHKorderStat
   
   Описание:
   Статистика заказов
   
   Код плагина взять из файла shk_orderstat.tpl.
   
   Системные события:
   OnSHKmodRenderTopLinks,OnSHKmodPagePrint
   
4. Нажмите "Сохранить". В модуле shopkeeper (Уравление заказами) появилась кнопка "Статистика заказов".
