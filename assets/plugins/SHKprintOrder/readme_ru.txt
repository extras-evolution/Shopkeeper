
SHKprintOrder 1.3 плагин для Shopkeeper >= 1.0

Печать информации о заказе.

----------------------------------------------

Установка:

1. Поместите папку SHKprintOrder в папку с плагинами - assets/plugins/.

2. В системе управления откройте "Элементы" -> "Управление элементами" -> "Плагины" -> "Создать плагин".

3. Название плагина:
   SHKprintOrder
   
   Описание:
   Печать информации о заказе
   
   Код плагина (php) взять из файла SHKprintOrder_plugin.tpl
   
   Конфигурация плагина:
   &print_text=Текст кнопки;string;Печать &tpl_path=Папка с шаблонами;string;assets/plugins/SHKprintOrder/chunks/ &print_tpl=Шаблон (чанк);string;@FILE:printOrder.tpl
   
   Системные события:
   OnSHKOrderDescRender
   
4. Открыть модуль "Управление заказами" (Shopkeeper) и нажать на ссылку "Подробности" с строке заказа.
   Внизу справа от кнопки "Назад" будет кнопка "Печать", после нажатия на которую появится окно предпросмотра печати.
